# このリポジトリについて
令和5年8月17日（木曜日）10時00分に県全域の3次元点群データを公開されるという「オープン・ナガサキ」点群データについて、その「自由に　使え」という思想に即して、コンプリートな三次元タイルを作成してみるプロジェクトです。

まずは issues でアイディアを練っています。

See: [Issues](https://github.com/optgeo/ngs)
